1.1.2 Aster
============

This is the 2nd patch release in the Aster 1.1 series.

Notable changes
----------------

* Add clean_ceph.sh script for deleting ceph.
* Add metallb installation option.
* Nodes except a deployer do not need to put host information in /etc/hosts.
